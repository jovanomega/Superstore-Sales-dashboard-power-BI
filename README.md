# Superstore-Sales-dashboard-power-BI
An interactive Power BI dashboard built using the Superstore dataset to analyze retail sales performance. The project includes data cleaning with Power Query, creation of DAX measures for key KPIs, and visualization of trends across regions, customer segments, and product categories to derive actionable business insights.
# 📊 Superstore Sales Performance Dashboard

## 🚀 Project Overview
This project focuses on analyzing retail sales data using Power BI to uncover key business insights and improve decision-making.

## 📌 Objectives
- Track key KPIs such as Sales, Profit, Orders, and Discounts
- Analyze sales trends over time
- Identify top-performing products and categories
- Compare performance across regions and customer segments

## 🛠️ Tools & Technologies
- Power BI
- DAX (Data Analysis Expressions)
- Power Query (ETL)
- Data Visualization

## 📊 Dashboard Preview

## 💡 Key Insights
- Technology category generates the highest revenue
- West region leads in total sales
- Some sub-categories have high sales but low profit
- Sales show consistent growth over time
